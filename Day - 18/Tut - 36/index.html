Understanding CSS Transforms

CSS Transforms allow you to move, rotate, scale, and skew elements. Hover over the boxes below to see different transform effects.

1. Translate

Hover Me

The translate() function moves an element from its current position.

2. Rotate

Hover Me

The rotate() function rotates an element clockwise or counter-clockwise.

3. Scale

Hover Me

The scale() function increases or decreases the size of an element.

4. Skew

Hover Me

The skew() function tilts an element along the X and/or Y axis.

5. 3D Rotate

Hover Me

CSS also supports 3D transforms like rotateX(), rotateY(), and rotateZ().

6. Multiple Transforms

Hover Me

You can combine multiple transform functions in one property.
